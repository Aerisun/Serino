# Cosmic Canvas Frontend

Frontend scaffold based on `Vite + React + TypeScript + Tailwind`.

## Scripts

- `pnpm install`
- `pnpm dev`
- `pnpm build`
- `pnpm lint`

## Structure

- `src/pages`: route pages
- `src/components`: shared UI blocks used by pages
- `src/contexts`: app-wide state such as theme
- `public`: static assets
